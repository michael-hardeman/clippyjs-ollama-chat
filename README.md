# clippyjs-ollama-chat

Have you ever wanted your AI experience to be more aesthetic and personable?
Have you ever missed the friendly paperclip who's alway here to help?

This repo can help with that.

Chat with a local LLM in an aesthetic windows 95 inspired UI.

Play DnD with merlin!
Have clippy help with your letter!
Hear jokes from bonzi buddy!

![Demo](./demo.png)

## How to use

Have ollama installed: https://ollama.com/

Change the model to whatever model you want to chat with.

Open `./index.html` in your browser.
